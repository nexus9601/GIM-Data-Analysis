# GIM Data Analysis

A data analysis project based on the Green India Mission (GIM) dataset. This project focuses on cleaning the dataset, exploring trends, and visualizing state-wise and yearly data using Python.

## Project Structure

```
GIM-Data-Analysis/
│
├── data/
│   ├── raw_data.csv
│   └── cleaned_gim_data.csv
│
├── notebook/
│   └── GIM_Trends_Data_Analysis.ipynb
│
└── README.md
```

## Objectives

- Clean and preprocess the raw dataset.
- Analyze funding and physical area covered under GIM.
- Compare state-wise performance.
- Study yearly trends.
- Present insights using visualizations.

## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed

- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- State-wise Analysis
- Year-wise Trend Analysis
- Summary Statistics

## Visualizations

The notebook includes:

- State-wise Line Charts
- Year-wise Trend Analysis
- Multiple Subplots for State Comparisons

## Dataset

The repository contains:

- **raw_data.csv** – Original dataset
- **cleaned_gim_data.csv** – Processed dataset used for analysis

## Getting Started

Clone the repository:

```bash
git clone https://github.com/<your-username>/GIM-Data-Analysis.git
```

Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn notebook
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```
notebook/GIM_Trends_Data_Analysis.ipynb
```
